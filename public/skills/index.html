    프로젝트 명
    아이알피 전자출결 시스템(어린이집) 관리자 페이지 구현 및 유지보수
    프로젝트 설명
    비콘과 키오스크를 이용하여 어린이집에서 아이들이 등원, 하원을 했을 경우 출석시간을 체크하여 보육통합시스템으로 전송하는 프로젝트
    성과
    • 아이알피 사업은 출시 1년만에 20억원이라는 매출 달성!
    • 어린이집 보육시스템 만족도 조사 1위
    본인 경험
    • 반복적으로 Import하는 팝업에 대해 전역 메서드로 구현을 이용하여 반복 작업 개선
    • 한달에 한번 수동으로 백업하는 방식에서 Scheduler를 이용하여 매일 저녁에 백업하도록 개선 (백업시간 6시간 -> 1~3분으로 단축)
    • 사업 초반 키오스크를 셋팅할 때 .bat 파일로 자동 셋팅 프로그램을 구현하여 작업속도 단축
    직무 내용
    • 프론트 페이지를 Putty와 Git을 사용하여 AWS 서버에 배포
    • 관리자페이지의 원아의 출결 데이터를 조회 및 디버깅 할 수 있는 페이지 개발
    • 알림톡 업체의 API를 이용해 하루 만개 가량의 등원, 하원 알림톡 전송 시스템 구축
    • 프론트에서 필요한 API를 백엔드에서 Laravel ORM을 이용하여 API 구축
    • CS팀에서의 관리자 페이지를 사용하면서 불편한 점을 물어보고 그 불편한 점을 개선하기 위해 고민 후 기능 개선
